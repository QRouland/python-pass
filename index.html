python-pass

python bindings

python-pass provides python bindings for pass. The PasswordStore class provides methods for most operations that one would want to do on a password store. The API is documented at pypass.readthedocs.org.

pypass

pypass is a python implementation of the pass command. There are a few differences between pass and pypass.

  • pypass lets you Init from an existing git repository
  • pypass does not depend on tree

For a complete manual of pypass, you may read the manpage on readthedocs.
